# Bitplus Signal Fullscreen Deck Prompts

These prompts are the Image 2.0 visual direction for the 16:10 fullscreen
hackathon pitch deck. The shipped PPTX uses editable slide text and vector-style
visual plates so the deck remains reliable in PowerPoint; these prompts can be
used to regenerate or replace each slide background with raster artwork.

## Slide 1: Cover

Use case: productivity-visual
Asset type: fullscreen 16:10 PowerPoint cover background
Primary request: Create a polished retro-modern cover image for Bitplus Signal,
a public-signal outreach intelligence tool for BTC++ Toronto.
Scene/backdrop: split visual metaphor, left side a tasteful mid-century route
book and doorstep field notes, right side a modern abstract conference signal
console with ranked opportunity cards and connection lines.
Style/medium: high-quality editorial illustration, cinematic landscape, warm
vintage print texture blended with crisp modern dashboard UI shapes.
Composition/framing: 16:10 landscape, strong empty space on the left for
editable text overlay.
Constraints: no readable text, no logos, no surveillance imagery, no dollar
signs, no watermark, no crypto coin symbols.

## Slide 2: The Problem

three short problem statements.

tracking imagery, no readable microtext.
